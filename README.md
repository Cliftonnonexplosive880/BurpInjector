## 🖥️ What BurpInjector Does

BurpInjector is a Burp Suite plugin for checking two common places where SQL injection issues may show up:

- URL paths
- X-Forwarded-For headers, also called XFF headers

It also includes interface auto-collection, which helps you gather and track API data while you work.

This tool is made for users who want a simple way to inspect traffic and spot weak points in web apps.
